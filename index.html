How to Play Simon

Easy Mode

1. Watch the sequence of colors that Simon shows you

2. Repeat the sequence by clicking the colors in the same order

3. Each level adds one more color to the sequence

4. The full sequence is shown before you need to repeat it

Hard Mode

1. Watch the sequence carefully - only the new color is shown

2. You must remember the entire sequence from memory

3. Each level adds one more color to remember

4. Only the newest color is displayed each turn

Ultra Mode

1. Watch carefully - Simon shows 1-3 new colors each turn

2. You must remember the entire growing sequence

3. The sequence grows faster with multiple colors per turn

4. No repeats - you see each new set only once
